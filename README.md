# APPFINAL---LISTA-DE-COMPRAS

Último Aplicativo feito na aula de Programação para Dispositivos Móveis. 
Docente: Marcos Vasconcelos de Oliveira

Aqui estão algumas funcionalidades do App.

Funcionalidades:

1 - Utilizei o ListView para adicionar os produtos digitados
2 - Usei o EditText para o usuário digitar os produtos a serem adicionados
3 - Adicionei um botão para adicionar o produto digitado.
4 - Adicionei a função de limpar (Clear) a mensagem no EditText depois de inserirmos o produto na lista, para facilitar quando formos inserir outro produto.
